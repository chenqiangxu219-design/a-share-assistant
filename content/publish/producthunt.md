# ProductHunt Launch Post

## Tagline
Your personal A-share trading terminal — built solo with AI Agents.

## Description
A股助手 is a TradingView-style desktop app for Chinese stock market analysis. Real-time quotes from 5 data sources, 10+ technical indicators, AI-powered chat with tool-use capability, and strategy backtesting — all in one app.

Built by 1 developer + AI Agent team (CEO/Architect/Coder/Reviewer/QA/Content) in under 2 weeks.

## Tech Stack
- Go 1.26 backend (WebSocket push, SQLite)
- React 19 + Electron frontend
- Python AI service (akshare for data, oMLX for LLM)
- ~24,000 lines of code

## What Makes It Special
1. **AI Agent Team** — 6 specialized agents collaborating on PRD → code → review → release
2. **Real-time** — WebSocket push from 5 Chinese data sources (Sina, EastMoney, Tencent, iwencai, ithomer)
3. **AI Chat with Tools** — 9 financial tools callable by natural language ("show me MAIBO's technicals")
4. **Cross-platform** — macOS (arm64/x64), Windows, Linux
