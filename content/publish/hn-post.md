A professional real-time trading analysis tool for Chinese A-shares with a TradingView-style UI. Tech stack: Go 1.26 backend (Gin + WebSocket + SQLite), React 19 frontend with Electron, and a Python AI microservice (LangChain + local LLM).

Key features:
- Real-time quotes via WebSocket from 5 data sources (Sina, EastMoney, Tencent, etc.)
- Technical analysis with 10+ indicators (MA, EMA, MACD, RSI, KDJ, Bollinger Bands)
- AI chat assistant for news sentiment analysis and sector rotation tracking
- Portfolio simulation with P&L tracking and strategy backtesting

Built by 1 person + an AI Agent team (6 specialized agents: CEO, Architect, Coder, Reviewer, QA, Content). 24,000+ lines of code across Go, TypeScript, and Python. v1.0.0 released with macOS, Windows, and Linux binaries.
